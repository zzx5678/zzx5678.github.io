쇼핑몰
- 로그인 -
스프링 시큐리티를 통한 보완성을 강화하여
간단한 안전한 로그인 기능을 구현했습니다.
아이디 저장 / 자동 로그인 기능은 쿠키를 통해
구현했으며, 웹 페이지를 불러 올 때 쿠키가 있으면
아이디 저장 / 자동 로그인이 되게 했습니다.
- 마이 페이지 -
회원 정보 수정 기능으로 간단히
비밀번호 및 생일, 전화번호 등을 수정할 수 있습니다.
주문 내역 확인 기능으로 간단히
주문 내역을 확인 할 수 있습니다.
그 외에는 로그아웃 기능과 메인화면 혹은 상품 페이지로
이동할 수 있는 기능이 있습니다.
- 주문 내역 -
비회원일 경우에는 배송 정보 입력 시,
사용한 전화번호와 비회원 주문 비밀번호를 통해 주문 내역과 배송 정보를 확인할 수 있습니다.
회원일 경우에는 별도의 조회 과정 없이
바로 주문 내역과 배송 정보을 열람할 수 있습니다.
- 프로젝트 후기 -
가볍게 만들었지만 필요한 기능을 모두 구현한 점이
만족스러웠으며, 다만, 진행하면서 더 완성도 있게 만들고
싶었으나 팀 프로젝트 준비로 그러지 못해 아쉬움이
조금 남습니다.
