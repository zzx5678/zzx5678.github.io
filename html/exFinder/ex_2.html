exFinder
프로젝트 테이블
총 12개의 데이터베이스 테이블을 사용했습니다.
그 중 핵심 테이블은 '통화' 와 '유저' 테이블입니다.
- 통화 테이블은
통화 코드, 통화 명, 통화 국가의 3개 컬럼이 있으며,
통화 코드를 기본키로 설정했고,
exFinder에서 제공하는 20가지 통화를 저장했습니다.
- 유저 테이블은
유저 아이디, 패스워드, 이름 등 12개의 컬럼이 있으며,
유저 아이디를 기본키로 설정했습니다.
이 두 기본키를 바탕으로
    나머지 10개 테이블과의 외래키를 설정했습니다.
나의 개인 기여
- DB와 구글 차트를 활용한 환율 그래프 제작
- DB와 구글 차트를 활용한 오늘의 세계 증시 제작
- Naver SMTP 서버를 이용한 이메일 인증 기능 구현
- Coolsms 서비스를 이용한 문자 인증 기능 구현
- 아이디 저장 및 자동 로그인 기능 구현
- 아이디 / 비밀번호 찾기 기능 구현
- web crawling을 이용한 환율 관련 뉴스 실시간 제공
- 마이 페이지의 계정 관리 구현
